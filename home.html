1. github란?
Github = git + hub
git: 파일의 변천사를 저장하는 역할, hub:중심지, 모여있는 곳
결론: git은 코드의 변화를 기록, github는 git과 코드를 저장하고 관리하는 서비스, 개발자의 포트폴리오의 역할

깃 명령어

1. git init 2. git remote add origin 레포지토리 주소 3. git add . 4. git commit -m "원하는 내용" 5.git push origin master
